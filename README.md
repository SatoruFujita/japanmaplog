JapanMaplog
====

Overview
写真投稿をメールで行うことにより、
記事投稿できる機能と、
投稿した記事に応じて地図に色を塗ることができる機能を持ったアプリです。

## Description
このアプリはメールで新規登録時のアドレスに文章と写真を投稿すると、
アップロードされ、表示されるようになります。
また、その際に画像の緯度経度情報を読み取り、
該当する都道府県をもとにトップ画面の地図に色を塗ることができるアプリです。
